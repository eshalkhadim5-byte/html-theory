## Q1: What are semantic HTML tags?
Tags that clearly describe their meaning. Examples: `<header>`, `<nav>`, `<article>`, `<footer>`.

## Q2: Difference between div and span?
div = block element (full width, new line). span = inline element (only needed width).

## Q3: Block-level vs inline elements? 
Block: starts new line, full width. Examples: `<div>`, `<p>`, `<h1>`.
Inline: no new line, needed width. Examples: `<span>`, `<a>`, `<img>`.

## Q4: Purpose of DOCTYPE declaration?
Tells browser which HTML version to use. Without it, browser goes into "quirks mode".

## Q5: Difference between id and class?
id = unique, identifies one element. class = reusable, groups multiple elements.

## Q6: How to create a form? Common input types?
Use `<form>` tag. Inputs: text, email, password, radio, checkbox, submit, date.

## Q7: What are meta tags?
Tags inside `<head>` that don't display but inform browser/search engines. Used for charset, SEO, viewport.

## Q8: Purpose of alt attribute in <img>?
Shows text if image fails to load. Used by screen readers. Helps SEO.

## Q9: How to make image clickable?
Wrap `<img>` inside `<a>` tag.

## Q10: Difference between JPG, PNG, SVG, WebP?
JPG = small photos, no transparency. PNG = graphics, supports transparency. SVG = scalable vector. WebP = modern, smallest size.

## Q11: Semantic HTML5 tags?
`<header>` = top, `<footer>` = bottom, `<section>` = related content, `<article>` = independent content.

## Q12: Difference between script, async, defer?
script = blocks parsing. async = downloads parallel, runs immediately. defer = downloads parallel, runs after HTML.

## Q13: Embed audio/video in HTML5?
Use `<audio>` and `<video>` tags with control attribute for play/pause buttons.

## Q14: Relative vs absolute paths?
Relative = from current file (images/pic.jpg). Absolute = full URL (https://site.com/images/pic.jpg).

## Q15: What are data attributes (data-*)?
Custom attributes to store extra data. Format: `data-name="value"`. Access via `element.dataset.name`.

## Q16: Purpose of viewport meta tag?
Makes page responsive on mobile devices.

## Q17: How to improve SEO using HTML?
Use semantic tags, proper `<title>`, meta description, alt text, heading hierarchy (h1>h2>h3), mobile-friendly.

## Q18: Accessibility best practices?
Add alt text, use `<label>`, enable keyboard navigation, good color contrast, use semantic HTML.

## Q19: Difference between strong vs b, em vs i?
strong/em = semantic (screen readers stress). b/i = visual only (no meaning).
